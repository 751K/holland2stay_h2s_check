# Holland2StayNotifier
## Installation:
```bash
python3 -m venv venv
source venv/bin/activate
cd h2snotifier

cp config_example.json config.json 
cp env_example .env 
# Update config.json and .env with you desired chat group

pip install -r requirements.txt

# Test one-time
python main.py
#
```
